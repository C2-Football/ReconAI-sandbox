Depth Below The AI Layer

Scout should be fast first, but never shallow.

The home screen can stay decision-led while each serious fantasy workflow opens into a focused mobile studio. The app should let users build trades, run dynamic mock drafts, manage rookie boards, and work waivers with enough depth to trust the output.

The model: Scout starts with a recommendation, then lets the user inspect assumptions, adjust inputs, save the artifact, and continue the deeper version in War Room.

Team And League Portfolio

Scout needs to understand the manager, not just one roster.

Fantasy players want to inspect their own teams deeply and run multiple leagues without bouncing through a hard switcher. Scout should have a personal portfolio layer: every team, every urgent move, every saved artifact, one command context.

The operating idea: choose a league when detail matters, but let the AI reason across every team by default. “Where am I exposed?” should work across dynasty, keeper, and redraft.
